# Security Policy

## Reporting Security Vulnerabilities

Please do **not** open a public issue for security vulnerabilities.

Instead, use [GitHub's private vulnerability reporting](https://github.com/nathanjohnpayne/tadlockpsychiatry/security/advisories/new) or email security concerns directly.

## Scope

This repository is maintained by a solo developer with AI agent assistance. Security reports are reviewed within 48 hours.

## Supported Versions

Only the latest version on the `main` branch is supported.

## Security Measures

- Secret scanning with push protection is enabled
- Dependabot monitors dependencies for known vulnerabilities
- All changes require pull request review before merge
