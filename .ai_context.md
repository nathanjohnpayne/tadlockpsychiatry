## Deploy Tooling

This repo follows the AI Agent Tooling Standard's keyless deployment model.
Human maintainers authenticate through a shared 1Password-backed GCP ADC
source credential, then deploy through short-lived service account
impersonation. CI prefers Workload Identity Federation.

### Scripts

| Script | Canonical source | Installed to | Purpose |
|---|---|---|---|
| `gcloud` | `scripts/gcloud/gcloud` | `~/.local/bin/` | Local wrapper that refreshes access tokens from a 1Password-backed source credential |
| `op-firebase-deploy` | `scripts/firebase/op-firebase-deploy` | `~/.local/bin/` | Deploy by impersonating the deployer service account |
| `op-firebase-setup` | `scripts/firebase/op-firebase-setup` | `~/.local/bin/` | One-time setup: creates deployer SA, grants IAM roles, configures impersonation |

### Credential Model

### Usage

```bash
# One-time per project
op-firebase-setup tadlockpsychiatry

# Deploy (auto-detects project from .firebaserc)
op-firebase-deploy

# Deploy specific targets
op-firebase-deploy --only hosting
```
